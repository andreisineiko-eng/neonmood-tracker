# NeonMood — Трекер заказов

Трекер статуса заказов для клиентов рекламного агентства.

## 🚀 Быстрый деплой на Vercel (бесплатно)

### Шаг 1 — Загрузите на GitHub
1. Зайдите на [github.com](https://github.com) → создайте аккаунт (если нет)
2. Нажмите **"New repository"** → назовите `neonmood-tracker`
3. Загрузите все файлы этой папки

### Шаг 2 — Разместите на Vercel
1. Зайдите на [vercel.com](https://vercel.com) → войдите через GitHub
2. Нажмите **"Add New Project"**
3. Выберите репозиторий `neonmood-tracker`
4. Настройки оставьте по умолчанию → нажмите **Deploy**
5. Готово! Вы получите ссылку вида `neonmood-tracker.vercel.app`

## 💻 Локальный запуск

```bash
npm install
npm run dev
```

Данные хранятся в `localStorage` браузера. Это значит:
- База видна только в том браузере, где вы её заполняли
